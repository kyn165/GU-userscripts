群馬大学生向けユーザースクリプト

このサイトでは、LMS (Moodle) や、教務システムなど、群馬大学の学生が利用するサイトをもっと便利で快適にするためのユーザースクリプト（拡張機能のようなもの）を配布しています。

これらのユーザースクリプトを使用することによって、たまに不便を感じたり、使いにくいと感じることもある各種サイトの利用が、より快適になると思います。

ユーザースクリプトで追加の通信を行うことはありませんので、大学側にも負荷がかかることはないと思いますし、これらのユーザースクリプトを利用したことで、損害などが起こることはないとは思いますが、万が一発生した際には、開発者や管理者は責任を負いかねますので、あくまで自己責任の範囲内でご活用ください。

このサイトやユーザースクリプトは、大学や総合情報メディアセンターと提携していませんし、また承認されていません。

これらのユーザースクリプトをインストールして使うには、ユーザースクリプトを実行できる拡張機能をインストールする必要があります。まだ入れていなければ、Tampermonkey をお勧めします。

動作環境は、Chrome か Edge の最新版です。これらの環境なら動作するはずです。
